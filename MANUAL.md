## 1. Πώς μπαίνω

### Διεύθυνση

Άνοιξε browser (Chrome / Safari / Firefox) και πήγαινε στη διεύθυνση:

👉 **https://www.kokalamarias.gr/admin/**

### Σύνδεση

1. Θα δεις το λογότυπο του ΚΟΚ και ένα κουμπί **«Login with GitHub»**
2. Πάτα το κουμπί
3. Ανοίγει ένα παράθυρο GitHub. Συνδέσου με τον λογαριασμό του ομίλου: **`KOKalamarias`**
4. Την πρώτη φορά θα σου ζητήσει **«Authorize»** — πάτα το
5. Το παράθυρο κλείνει αυτόματα και μπαίνεις στο CMS

⚠️ **Σημαντικό**: Μόνο όσοι έχουν πρόσβαση στον λογαριασμό **KOKalamarias** στο GitHub μπορούν να μπουν. Αν θες να δώσεις πρόσβαση σε άλλο άτομο, μοιράσου τον κωδικό του GitHub λογαριασμού (ή μίλα στον υπεύθυνο τεχνικής υποστήριξης).

## 2. Η εικόνα του CMS

Όταν μπαίνεις βλέπεις:

- **Πάνω-αριστερά**: «Contents» (περιεχόμενο) και «Media» (φωτογραφίες)

  - 📰 **Νέα & Ανακοινώσεις** — αγωνιστικά/αθλητικά άρθρα

  - 🏊‍♂️ **Τμήματα** — επίπεδα κολύμβησης
  - 📝 **Κείμενα Σελίδων** — κεντρικά κείμενα (Hero, About, Επικοινωνία κλπ)

- **Πάνω-δεξιά**: το όνομα του ομίλου και το avatar — από εκεί κάνεις **logout**

## 3. Νέο Άρθρο

### Βήμα-βήμα

**1)** Πάτα **«📰 Νέα & Ανακοινώσεις»** στα αριστερά.

**2)** Πάτα **«Λίστα Άρθρων»** στο κέντρο.

**3)** Θα ανοίξει ο editor. Στην ενότητα **«Άρθρα (τα νεότερα πρώτα)»** βλέπεις όλα τα υπάρχοντα άρθρα.

**4)** Κάτω από τη λίστα ή σε κάποια άκρη, υπάρχει κουμπί **«Add Άρθρα»** (ή «+ Προσθήκη»). Πάτα το.

**5)** Εμφανίζεται **νέα κενή καταχώρηση**. Συμπλήρωσε τα πεδία:

| Πεδίο | Τι βάζω |
|---|---|
| **Τίτλος** | Π.χ. *Νίκη στο Πανελλήνιο 2026* |
| **Τίτλος (Αγγλικά)** | Π.χ. *Victory at the 2026 Greek Championship* (προαιρετικό αλλά συνιστάται) |
| **URL slug** | Π.χ. `niki-panellinio-2026` — δες ενότητα [Slugs](#-τι-είναι-το-url-slug) παρακάτω |
| **Ημερομηνία** | Επίλεξε από ημερολόγιο. Είναι η ημερομηνία του γεγονότος |
| **Ετικέτα ημερομηνίας** | Π.χ. *Καλαμαριά · Μάιος 2026* (όπως θες να εμφανίζεται ως μικρή ετικέτα) |
| **Ετικέτα ημερομηνίας (Αγγλικά)** | Π.χ. *Kalamaria · May 2026* |
| **Κατηγορία** | Επίλεξε από: Πρωτάθλημα, Παγκόσμιο, Ευρωπαϊκό, Ρεκόρ, Εθνικές Ομάδες, Εγγραφές, Εκδήλωση, Κοινωνικό |
| **Κύρια φωτογραφία** | Προαιρετικό. Πάτα **«Choose Image»** για να ανεβάσεις/επιλέξεις φωτό |
| **Video URL** | Προαιρετικό. URL από YouTube ή Vimeo. Δες ενότητα [Videos](#-βίντεο-σε-άρθρα) παρακάτω |
| **Σύντομη περίληψη** | 1-2 προτάσεις που θα φαίνονται στη λίστα νέων |
| **Σύντομη περίληψη (Αγγλικά)** | Η αγγλική μετάφραση |
| **Πλήρες κείμενο** | Το βασικό κείμενο του άρθρου (markdown) |
| **Πλήρες κείμενο (Αγγλικά)** | Η αγγλική έκδοση |
| **Featured** | ⭐ Αν θες αυτό να φανεί ως μεγάλη κάρτα με φωτογραφία στην κορυφή. **Μόνο 1-2 άρθρα** ταυτόχρονα featured! |
| **Δημοσιευμένο** | ✅ Άστο τσεκαρισμένο για να φαίνεται. Αν είναι ξετσέκαρο, κρύβεται από το site |

**6)** Σύρε το νέο άρθρο **στην κορυφή της λίστας** (αν θέλεις να εμφανίζεται πρώτο). Για να σύρεις, πάτα και κράτα τη δεξιά πλευρά της εγγραφής όπου υπάρχει εικονίδιο ⋮⋮ (κουκίδες).

**7)** Πάτα **«Publish»** πάνω-δεξιά → **«Publish now»**. (Δες ενότητα [10](#10-publish) για λεπτομέρειες.)

**8)** Σε ~1-2 λεπτά οι αλλαγές θα εμφανιστούν στο **https://www.kokalamarias.gr/news**.

## 4. Επεξεργασία Άρθρου

**1)** Πάτα **«📰 Νέα & Ανακοινώσεις»** → **«Λίστα Άρθρων»**.

**2)** Βρες το άρθρο που θες να αλλάξεις. Πάτα πάνω στον τίτλο για να ξεδιπλωθεί.

**3)** Άλλαξε όποια πεδία θες.

**4)** **Publish** → **Publish now**.

### Διαγραφή άρθρου

Στο πλάι κάθε άρθρου υπάρχει εικονίδιο **κάδου 🗑** ή «X». Πάτα το και επιβεβαίωσε.

⚠️ Η διαγραφή είναι μόνιμη μετά το Publish! Εναλλακτικά μπορείς να ξετσεκάρεις το **«Δημοσιευμένο»** ώστε να κρυφτεί χωρίς να χαθεί.

## 5. Φωτογραφίες

### Πώς ανεβάζω καινούργια φωτό

**Μέθοδος Α — Όταν φτιάχνεις άρθρο/εγγραφή:**
1. Στο πεδίο **«Κύρια φωτογραφία»** πάτα **«Choose Image»**
2. Στο παράθυρο που ανοίγει, πάτα **«Upload»** πάνω-δεξιά
3. Επίλεξε εικόνα από τον υπολογιστή σου
4. Μόλις ανέβει, **πάτα πάνω της** και πάτα **«Choose Selected»**

**Μέθοδος Β — Γενικά (όλες οι φωτογραφίες μαζί):**
1. Πάτα **«Media»** πάνω-αριστερά
2. Πάτα **«Upload»** πάνω-δεξιά
3. Επίλεξε μία ή πολλές εικόνες
4. Έτοιμο — οι φωτό σώζονται στο φάκελο `images/uploads/`

### Συμβουλές για φωτογραφίες

**Για κύριες φωτό άρθρων (banner)**
- ✅ Μέγεθος: ιδανικά **1600×900 px** (οριζόντιο, αναλογία 16:9)
- ✅ Format: **.jpg** για φωτογραφίες, **.png** για graphics/λογότυπα
- ✅ Συμπίεση: όχι πάνω από **500 KB** (https://tinypng.com)
- ❌ Όχι κάθετες φωτογραφίες — θα κοπούν

**Για avatar αθλητή (στρογγυλό)**
- ✅ Μέγεθος: ιδανικά **600×600 px** (τετράγωνο)
- ✅ Headshot (κεφάλι + ώμοι), πρόσωπο στο κέντρο
- ✅ Format: **.jpg** προτιμότερο
- ✅ Μέγεθος αρχείου: < **300 KB**
- ❌ Όχι πλήρες σώμα (το πρόσωπο θα φαίνεται σαν κουκίδα)
- ❌ Όχι ομαδικές — μόνο 1 άτομο
- ❌ Όχι σκούρο φόντο (χάνεται μέσα στην κάρτα)

**Πώς φτιάχνω τετράγωνη φωτό**
- **iPhone**: Φωτογραφίες → Edit → Crop → Aspect Ratio «Square»
- **Mac Preview**: Tools → Crop → Aspect Ratio → Square
- **Online**: https://www.iloveimg.com/crop-image

## 8. Τμήματα

**1)** Πάτα **«🏊‍♂️ Τμήματα»** → **«Τμήματα Κολύμβησης»**.

**2)** Για κάθε τμήμα:
   - **Όνομα Τμήματος** + **Αγγλικά**
   - **URL slug** (π.χ. `academy`)
   - **Ηλικίες**: π.χ. *4-7 ετών* / *Ages 4-7*
   - **Περιγραφή** + **Αγγλικά** — σύντομη παράγραφος (όχι FAQ!)
   - **Πρόγραμμα**: π.χ. *Δευτέρα-Τετάρτη-Παρασκευή 17:00*
   - **Παρουσίαση Προπονητή** (προαιρετικό): π.χ. *«Απαντά ο Στράτος Μαλαμαδάκης — Υπεύθυνος προπονητής εκμάθησης»*
   - **Ερωτήσεις & Απαντήσεις (FAQ)** — δες αναλυτικά παρακάτω ⬇
   - **Featured**: ✓ μόνο αν είναι το αγωνιστικό (μεγαλύτερη κάρτα)

### Ερωτήσεις & Απαντήσεις (FAQ)

Κάθε τμήμα μπορεί να έχει τη δικιά του λίστα από συχνές ερωτήσεις. Εμφανίζονται στη σελίδα του τμήματος (π.χ. `/teams/academy`) ως **όμορφο accordion** — ο επισκέπτης πατάει την ερώτηση και ξεδιπλώνεται η απάντηση.

**Πώς προσθέτω ερώτηση:**

1. Στην επεξεργασία τμήματος, βρες την ενότητα **«Ερωτήσεις & Απαντήσεις (FAQ)»**
2. Πάτα **«+ Add Ερωτήσεις...»**
3. Συμπλήρωσε:
   - **Κατηγορία** (προαιρετική): π.χ. `Κλασικής Κολύμβησης` ή `Τεχνικής Κολύμβησης`. Αν τη βάλεις, οι ερωτήσεις θα ομαδοποιηθούν με υπότιτλο. Αν την αφήσεις κενή, όλες οι ερωτήσεις εμφανίζονται μαζί.
   - **Ερώτηση** (ελληνικά)
   - **Ερώτηση (Αγγλικά)** — προαιρετική
   - **Απάντηση** (ελληνικά)
   - **Απάντηση (Αγγλικά)** — προαιρετική
4. Επανάλαβε για κάθε ερώτηση
5. Publish

**Tips:**
- Κάνε τις ερωτήσεις **σύντομες και ξεκάθαρες** (όπως θα ρωτούσε κάποιος γονιός)
- Οι απαντήσεις μπορούν να είναι 1-4 προτάσεις
- Σύρε τις ερωτήσεις πάνω-κάτω για να αλλάξεις σειρά
- Για να **διαγράψεις** μια ερώτηση: εικονίδιο κάδου δίπλα της

## 10. Publish

### Πώς γίνεται δημοσίευση

1. Αφού ολοκληρώσεις τις αλλαγές, πάτα **«Publish»** πάνω-δεξιά
2. Στο μενού επίλεξε **«Publish now»**
3. Θα φύγεις από τη φόρμα και θα γυρίσεις στη λίστα
4. **Σε ~1-2 λεπτά** οι αλλαγές εμφανίζονται στο πραγματικό site

### Γιατί καθυστερεί 1-2 λεπτά;

Επειδή κάθε αλλαγή τσεκαρισμένη με Publish:
- Αποθηκεύεται στο GitHub (το αποθετήριο του site)
- Το Cloudflare κάνει αυτόματο rebuild του site
- Όταν τελειώσει το build → οι αλλαγές γίνονται live

### Save vs Publish

- **Save** (αν εμφανιστεί): απλώς σώζει τις αλλαγές προσωρινά, **δεν** τις δημοσιεύει
- **Publish**: σώζει ΚΑΙ δημοσιεύει

## 9b. Κοινωνικές Δράσεις

Δράσεις κοινωνικής ευαισθητοποίησης (Heart Swim, Safe Water Sports, σχολικές επισκέψεις, Παραθινούπολη κλπ).

**Λειτουργεί ακριβώς όπως τα Νέα** — απλά δημοσιεύεται στη σελίδα `/social` αντί `/news`.

### Πώς προσθέτω νέα δράση

**1)** Πάτα **«🤝 Κοινωνικές Δράσεις»** στα αριστερά
**2)** Πάτα **«Λίστα Δράσεων»**
**3)** Πάτα **«+ Add Δράσεις»**
**4)** Συμπληρώστε όπως ένα κανονικό άρθρο:
   - Τίτλος + αγγλικά
   - **URL slug** (π.χ. `1o-kalamaria-heart-swim`)
   - Ημερομηνία + ετικέτα τοποθεσίας
   - **Κατηγορία**: Φιλανθρωπία / Safe Water Sports / Σχολικές Επισκέψεις / Παραλιακές Δράσεις / Οικογένεια Κ.Ο.Κ. / Άλλο
   - Κύρια φωτογραφία (προαιρετικό)
   - Σύντομη περίληψη + αγγλικά
   - Πλήρες κείμενο + αγγλικά
   - Featured / Δημοσιευμένο

**5)** **Publish → Publish now**

Η δράση εμφανίζεται:
- Στη λίστα: `https://www.kokalamarias.gr/social`
- Με δικό της URL: `https://www.kokalamarias.gr/social/<slug>` — βολικό για μοιρασιά σε social media

### Πότε χρησιμοποιώ «Νέα» και πότε «Κοινωνικές Δράσεις»;

## 🎬 Βίντεο σε άρθρα

Μπορείς να ενσωματώσεις βίντεο σε κάθε άρθρο (Νέα ή Κοινωνικές Δράσεις) χρησιμοποιώντας **YouTube** ή **Vimeo**.

### Πώς γίνεται — 3 βήματα

**1)** Ανέβασε το βίντεο στο **YouTube** (ή Vimeo). Αν δεν έχεις λογαριασμό YouTube του ΚΟΚ, δες tip παρακάτω.

**2)** Αντίγραψε το URL του βίντεο από τη μπάρα διεύθυνσης. Δεκτές μορφές:
- `https://www.youtube.com/watch?v=abc123`
- `https://youtu.be/abc123`
- `https://www.youtube.com/shorts/abc123`
- `https://vimeo.com/123456`

**3)** Επικόλλησε το URL στο πεδίο **«Video URL»** του άρθρου στο CMS.

Δεν χρειάζεται να ξέρεις τίποτα από embed codes ή HTML. Το site αναγνωρίζει αυτόματα τη μορφή και εμφανίζει σωστά τον player.

### Πού εμφανίζεται

- Στη σελίδα του άρθρου (`/news/...` ή `/social/...`) — μεγάλος responsive player κάτω από την περίληψη
- Στις κάρτες της λίστας (`/news` ή `/social`) — μικρό **«▶ Video»** badge δίπλα στην κατηγορία, ώστε ο επισκέπτης να ξέρει ότι υπάρχει βίντεο

### Tip: YouTube channel για τον ΚΟΚ

Αν δεν υπάρχει ήδη, δημιουργείστε **δωρεάν** κανάλι YouTube με το info@kokalamarias.gr λογαριασμό Google. Πλεονεκτήματα:
- Όλα τα βίντεο μαζεμένα σε ένα μέρος
- Δωρεάν, απεριόριστος χώρος
- Στατιστικά views/εγγραφές
- Δυνατότητα να βλέπει ο κόσμος και άλλα videos του ομίλου

### Παράδειγμα

Έστω ότι ανέβασες στο YouTube το βίντεο από το Heart Swim:
- URL: `https://www.youtube.com/watch?v=AbCdEfGh123`
- Στο CMS, στο άρθρο Heart Swim, πεδίο «Video URL»: επικολλάς το URL
- Publish → Σε 1-2 λεπτά εμφανίζεται player στη σελίδα `/social/1o-kalamaria-heart-swim`

### Τι δεν δουλεύει

❌ Direct ανέβασμα video file στο CMS — επειδή το site είναι στατικό, video αρχεία θα ήταν αργά και θα έκαιγαν bandwidth. Πάντα μέσω YouTube/Vimeo.
❌ Facebook/Instagram video — δεν υποστηρίζεται αυτόματο embed αυτή τη στιγμή.

## 12. Αγγλική έκδοση

Το site λειτουργεί σε **2 γλώσσες**: Ελληνικά (default) και Αγγλικά. Ο επισκέπτης κάνει εναλλαγή με το κουμπί **EL / EN** πάνω-δεξιά.

### Πώς λειτουργεί

Σε κάθε άρθρο/δράση/αθλητή/τμήμα υπάρχουν **διπλά πεδία**:

- **Τίτλος** + **Τίτλος (Αγγλικά)**
- **Σύντομη περίληψη** + **Σύντομη περίληψη (Αγγλικά)**
- **Πλήρες κείμενο** + **Πλήρες κείμενο (Αγγλικά)**
- κ.λπ.

### 3 σενάρια

**Σενάριο Α — Συμπληρώνω και τα δύο**
- Ελληνικά εμφανίζονται όταν ο επισκέπτης είναι σε EL mode
- Αγγλικά εμφανίζονται σε EN mode
- ✅ Ιδανικό

**Σενάριο Β — Συμπληρώνω μόνο ελληνικά**
- Σε EL mode εμφανίζονται κανονικά
- Σε EN mode εμφανίζονται **πάλι τα ελληνικά**, αλλά:
  - Σε ατομική σελίδα άρθρου εμφανίζεται κίτρινο banner: *«This article is only available in Greek. Your browser may offer automatic translation, or view in Google Translate.»*
  - Ο browser του επισκέπτη (Chrome, Edge, Safari) **προτείνει αυτόματα μετάφραση** χάρη σε τεχνική σήμανση που έχουμε στον κώδικα (`lang="el"`)
  - Εναλλακτικά, ο επισκέπτης πατάει το link στο banner και το βλέπει μέσω Google Translate
- ✅ Λειτουργεί χωρίς δικιά μας δουλειά, αλλά η ποιότητα μετάφρασης είναι αυτή του μηχανήματος

**Σενάριο Γ — Συμπληρώνω μόνο αγγλικά**
- Δεν θα συνιστούσαμε αυτό, αλλά λειτουργεί συμμετρικά (στα EL θα φαίνονται αγγλικά)

### 💡 Σύσταση

- Για **σημαντικά άρθρα/δράσεις** (π.χ. Παγκόσμια Πρωταθλήματα, μεγάλες φιλανθρωπικές δράσεις) → συμπλήρωνε **και τα δύο**
- Για **μικρές καθημερινές ανακοινώσεις** → ελληνικά αρκούν, αφήνεις τον browser να κάνει αυτόματη μετάφραση

### 🛠 Trick για εύκολη μετάφραση

1. Γράψε το ελληνικό κείμενο
2. Άνοιξε https://www.deepl.com/translator ή ChatGPT
3. Επικόλλησε το ελληνικό
4. Αντίγραψε την αγγλική μετάφραση
5. Επικόλλησε στο αντίστοιχο πεδίο `_en` στο CMS

**Συνολικός χρόνος: ~2 λεπτά παραπάνω ανά άρθρο** για επαγγελματική δίγλωσση δημοσίευση.

## 🔗 Τι είναι το URL slug

Το **URL slug** είναι ένα μικρό κομμάτι κειμένου που μπαίνει στο τέλος του URL για να δίνει σε κάθε άρθρο/αθλητή/τμήμα τη **δική του διεύθυνση**.

**Παράδειγμα:**
- Slug: `5o-protathlima`
- Τελική διεύθυνση: `https://www.kokalamarias.gr/news/5o-protathlima`

### Γιατί χρειάζεται;

✅ **Για να μοιράζεσαι συγκεκριμένο άρθρο** στα social media (όχι όλη τη λίστα νέων)
✅ **Για να βρίσκει σωστά το Google** το συγκεκριμένο περιεχόμενο (καλύτερο SEO)
✅ **Για να μένει σταθερή η διεύθυνση** ακόμα κι αν αλλάξεις τον τίτλο μετά

### Κανόνες για slugs

- **Μόνο μικρά αγγλικά γράμματα**: `a-z`
- **Αριθμοί επιτρέπονται**: `0-9`
- **Παύλες αντί για κενά**: `-`
- **Όχι κενά, ελληνικά, σύμβολα, κεφαλαία**

### Παραδείγματα

| Τίτλος άρθρου | ✅ Καλό slug | ❌ Λάθος slug |
|---|---|---|
| 5o Πανελλήνιο 2026 | `5o-panellinio-2026` | `5o Πανελλήνιο 2026` |
| Παγκόσμιο Ρεκόρ Τελιούση | `teliousi-paghosmio-rekor` | `Teliousi_Record!` |
| Εγγραφές 2026-27 | `eggrafes-2026-27` | `εγγραφές 2026/27` |
| Νέος Παίκτης | `neos-paiktis` | `Νέος Παίκτης` |

### Tips

- **Άστο σε άσπρο** αν δεν θες ξεχωριστή σελίδα (το άρθρο θα εμφανίζεται στη λίστα νέων κανονικά αλλά δεν θα έχει δικιά του διεύθυνση)
- **Άλλαξέ το** μόνο αν χρειαστεί. Αν αλλάξεις slug σε ένα άρθρο που έχεις ήδη μοιραστεί, το **παλιό link θα σπάσει**.
- **Κράτα το σύντομο** αλλά περιγραφικό. Όχι πάνω από 5-6 λέξεις με παύλες.
- Για **αθλητές**: συνηθίστε `firstname-lastname` ή σκέτο επίθετο (`panagiotidis`)

## 11. Troubleshooting

### «Δεν με αφήνει να μπω»
- Επιβεβαίωσε ότι συνδέεσαι με τον σωστό GitHub λογαριασμό (`KOKalamarias`)
- Αν είσαι σε άλλο GitHub λογαριασμό, κάνε logout από github.com και ξαναπροσπάθησε

### «Έγραψα κάτι αλλά δεν φαίνεται στο site»
- Επιβεβαίωσε ότι πάτησες **Publish now** (όχι μόνο Save)
- Περίμενε **2 λεπτά** πριν τσεκάρεις
- Καθάρισε cache του browser: ⌘+Shift+R στον Mac (ή Ctrl+F5 σε Windows)

### «Έχει βγει μήνυμα error / κίτρινο banner»
- Συνήθως πατώντας Publish ξανά λύνεται
- Αν επιμένει: βγες (Logout πάνω-δεξιά → avatar → Log out) και ξαναμπες

### «Έσβησα κάτι κατά λάθος»
- Όλες οι αλλαγές υπάρχουν στο **GitHub history**. Επικοινώνησε με τον τεχνικό υπεύθυνο για επαναφορά.

### «Θέλω να βγάλω άρθρο χωρίς να το διαγράψω»
- Στην επεξεργασία του άρθρου, **ξετσέκαρε** το πεδίο **«Δημοσιευμένο»** και Publish. Θα κρυφτεί από το site αλλά θα μείνει στη βάση.

### «Δεν φαίνεται η νέα φωτό που ανέβασα»
- Φωτογραφίες πάνω από **5 MB** μπορεί να αργήσουν ή να αποτύχουν. Συμπίεσε στο https://tinypng.com.
